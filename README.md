# 🌾 AI Based Farmer Query Support and Advisory System

An AI-powered web application developed using Django to assist farmers by providing quick, reliable, and intelligent agricultural advice. The system enables farmers to ask agriculture-related questions and receive AI-generated recommendations on crop management, farming techniques, pest control, fertilizers, irrigation, and other agricultural practices through an easy-to-use web interface.

---

## 📖 Overview

The **AI Based Farmer Query Support and Advisory System** is designed to support farmers by providing instant access to agricultural knowledge. Instead of waiting for expert consultation, farmers can submit their questions through the application and receive AI-assisted responses. The system aims to improve productivity, promote sustainable farming practices, and help farmers make informed decisions.

---

## ✨ Features

- 🤖 AI-powered agricultural query assistance
- 🌱 Crop management guidance
- 🐛 Pest and disease advisory
- 💧 Fertilizer and irrigation recommendations

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Based-Farmer-Query-Support-and-Advisory-System.git
```

### 2. Navigate to the Project

```bash
cd AI-Based-Farmer-Query-Support-and-Advisory-System
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Apply Migrations

```bash
python manage.py migrate
```

### 7. Start the Development Server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
